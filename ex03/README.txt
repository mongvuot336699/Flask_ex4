data: 
- file waiting_clients.csv: contains information about ID, name and wanted sim of clients who have not bought sim
- file all_clients.csv: contains information about ID, name and wanted sim of all clients (both waiting clients or clients who bought sim)
- file client0.csv, client1.csv, ... save information about ID, name, wanted sim and sim which client bought (if have) of client with id 0, 1, ... respectively 
- file sim.csv save all sim which were crawled based on waiting_clients.csv
